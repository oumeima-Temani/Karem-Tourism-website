# Tourism Company Website
Modern tourism company website built with Next.js and Tailwind CSS featuring responsive design and optimized user experience.

The project focuses on:
- Responsive UI design
- Modern user experience
- Tourism and hotel service presentation
- Optimized frontend performance

## Features
- Responsive homepage
- Hotel showcase
- Reservation pages
- Navigation system
- Optimized image handling
- Modern UI components

## Tech Stack

### Frontend
- Next.js
- React
- Tailwind CSS
- JavaScript


## Installation
npm install
npm run dev
